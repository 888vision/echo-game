# 星尘王座 · Stellar Throne

> 一个关于宇宙尺度下的失去、重逢与选择的策略叙事游戏。

## 概念

你是一颗刚刚死去的恒星的意识。你的恒星在数十亿年的燃烧后熄灭了，但在最后一刻，量子涨落赋予了你一个机会——将意识分散到宇宙的各个角落，像种子一样重新生长。

你要做的不是"征服世界"，而是**让你的意识重新覆盖整个宇宙**。

而在这个过程中，你会遇到其他"死去恒星"的意识——它们是竞争对手，也是曾经的爱人。

## 特性

- **3D 星图渲染** — Three.js WebGL 粒子系统，沉浸式宇宙探索
- **星光打字机对话** — 温柔的文字浮现效果，配合情感色彩变化
- **多分支叙事** — 融合 / 潮汐锁定 / 告别，每个选择改变你的命运
- **记忆碎片系统** — 你的每一次选择都成为永恒的星图印记
- **电影级开场** — 黑屏中的光点 → 恒星爆炸 → 星图显现
- **胶片颗粒质感** — 全局噪点叠加，营造沉浸氛围

## 技术栈

- **React 19** + **TypeScript** — 类型安全的前端框架
- **Three.js** — 3D 渲染引擎
- **Framer Motion** — 流畅的交互动画
- **Vite** — 极速构建工具

## 游戏流程

1. **开场动画** — 黑屏中的光点闪烁，点击后恒星爆炸，星图显现
2. **叙事文字** — 逐行浮现的故事，告诉你如何来到这里
3. **探索星图** — 拖拽旋转、滚轮缩放，点击星系查看详情
4. **殖民星系** — 选择要开发的星系，扩张你的意识场
5. **遇见意识** — 发现其他恒星意识，触发对话剧情
6. **做出选择** — 融合、锁定或告别，每个选择都有代价
7. **推进纪元** — 时间流逝，你的宇宙版图不断扩大

## 设计哲学

> 这不是一个关于胜利的游戏，而是一个关于选择和代价的游戏。

- 界面如抚摸星空般温柔
- 对话如初恋般真挚
- 选择如宇宙般沉重
- 结局如星光般独特

## 文件结构

```
src/
├── components/
│   ├── OpeningScene.tsx    # 开场动画（Three.js 粒子爆炸）
│   ├── OpeningText.tsx     # 叙事文字（打字机效果）
│   ├── StarField.tsx       # 3D 星图渲染
│   ├── HUD.tsx             # 顶部状态栏
│   ├── SidePanel.tsx       # 右侧信息面板
│   ├── StarDetail.tsx      # 星系详情弹窗
│   ├── DialoguePanel.tsx   # 对话面板（星光打字机）
│   └── Game.tsx            # 主游戏组件（已废弃，见 App.tsx）
├── hooks/
│   └── useGameState.ts     # 游戏状态管理 + 本地存档
├── styles/
│   ├── global.css          # 全局样式 + 动画关键帧
│   └── theme.ts            # 主题色板 + 字体配置
├── types/
│   └── game.ts             # TypeScript 类型定义
├── App.tsx                 # 根组件
└── main.tsx                # 入口文件
```
